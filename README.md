* **CSE 3311 - OBJECT-ORIENTED SOFTWARE ENGINEERING**  
    _This section contains a movie and TV show recommendation project, namely ***The Binge Spot***, I worked on with my team at the time. The website allows users to explore movies and TV shows through various sections, including Trending, Movies, TV Shows, Search, and our signature Movie Swipe feature. Our MovieSwipe feature sort of functions like TikTok, where a movie or TV show will be presented with information like the year, director/creator, where it can be streamed, cast, reviews, etc. Users can then decide if they want to swipe left or right to either skip to the next presented media or add it to their library. Users can also click on the media they find in other sections, view its information and details, and add it to their library from there as well._

* **CSE 4310 - FUNDAMENTALS OF COMPUTER VISION**  
    _This section contains assignments I worked on in computer vision. The first assignment is where I played with color spaces and data augmentation. I implemented RGB to HSV and HSV to RGB and optimized those two functions. I also implemented random cropping, patch extraction, resizing, color jitter, and image pyramids. The second assignment focused on the CIFAR-10 dataset, where I used SIFT and HOG to extract features, detect those keypoints and descriptors, match the descriptors and plot them, save the extracted features to a file, and then evaluate the SIFT and HOG features using an SVM model. I then wrote a little report about my findings that contained my general comments, results found, and some questions that needed to be answered about the assignment. For the third assignment, I worked on the Imagenette dataset and implemented a basic CNN, the All Convolutional Net, regularized the basic CNN, and used Transfer Learning, which was fine-tuned with the CIFAR-10 dataset, all using PyTorch Lightning. I then wrote a report that contained my findings and best weights. For the fourth and final assignment, I wrote a motion detector given an mp4 of a parking lot with moving traffic. I first wrote the Motion Detector class that detected motion and would draw bounding boxes on frames that would have motion. Then I implemented the Kalman Filter class, which filtered out noise and kept track of worthy movement in each frame based on a predict and update function. Multiple objects would be tracked through the PySide6 GUI, allowing the user to step frame by frame, or they can skip 60 frames ahead._
